# student-placement-analysis
This project performs **Exploratory Data Analysis (EDA)** on a student placement dataset to identify the academic, technical, and professional factors that influence placement outcomes.

The analysis focuses on understanding student performance, discovering placement patterns, identifying employability drivers, and providing data-driven recommendations to improve placement success.

## рџ“€ Key Insights

- Coding skills are the strongest predictor of placement success.
- Students with higher CGPA have better placement outcomes.
- Backlogs significantly reduce placement opportunities.
- Practical projects have a stronger impact than certifications alone.
- Internships consistently improve employability.
- Strong communication skills enhance placement chances when combined with technical skills.
- CSE and IT students show the highest placement rates.
- Students with no backlogs and multiple projects achieve the best placement outcomes.
